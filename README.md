# 🚂 Train Schedule Analysis and Interactive Route Enquiry System

## 📋 About This Project
This project performs comprehensive 
data analysis on Indian Railways 
train schedule dataset containing 
186,074 records and 11,113 unique 
trains using Python.

## 🏢 Internship
Company  : Sysslan IT Solutions
Domain   : Data Analytics
Project  : Train Schedule Analysis

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## 📊 Project Levels

### ✅ Level 1 - Basic Data Review
- Dataset overview
- Train routes listing
- Stops per train calculation
- Max and Min stops identified

### ✅ Level 2 - Data Processing
- Time fields standardization
- Journey duration calculation
- Route classification
- Station frequency count

### ✅ Level 3 - Data Quality Checks
- Missing values handled
- Duplicate records removed
- Station order verified
- Cleaned dataset saved

### ✅ Level 4 - Visualization
- Average journey duration chart
- High traffic stations chart
- Bar charts created
- Key observations summarized

### ✅ Level 5 - Advanced Analysis
- Pivot tables created
- Cross tabulations done
- Heatmap visualization
- Pie chart created

### ✅ Level 6 - Capstone System
- Interactive train enquiry system
- Search trains by source station
- Search trains by destination
- Journey distance calculator

## 📈 Key Findings
- Total Records  : 186,074
- Total Trains   : 11,113
- Total Stations : 1,234
- Route Types    : Short, Medium, Long
